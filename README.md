# Galactic Rush

> A fast-paced 3D endless runner set in space — dodge obstacles, collect diamonds, and survive as long as you can.

**[Play in Browser on itch.io](https://vs-studio-hub.itch.io/galactic-rush)** · Built with Unity · Solo Project

---

## What is it?

Galactic Rush is a space-themed endless runner where the speed keeps climbing and the obstacles keep coming. You dodge, jump, and slide your way through the galaxy while collecting diamonds and activating power-ups. The longer you survive, the harder it gets.

It plays in the browser — no download needed.

---

## Features

- Endless gameplay with progressively increasing difficulty
- Diamond collection and scoring system
- Power-ups: Magnet, Shield and x2 Score Multiplier
- Level-up and progression mechanics
- High score tracking
- Named player profile system — enter your name once, return anytime and pick up exactly where you left off

**Controls:**

| Key | Action |
|---|---|
| A / ← | Move Left |
| D / → | Move Right |
| Space | Jump |
| Alt | Slide |

---

## My Role

I built this entirely solo — every system, every line of code, every shader.

**Systems I built:**

- Endless procedural obstacle generation with dynamic difficulty scaling
- Player movement controller — left/right, jump, and slide with collision detection
- Power-up system — Magnet, Shield, and score multiplier with timed effects
- Diamond collection and scoring logic
- Level-up and progression system
- Player identity system — players enter a name on first launch; the game stores a named profile and fully restores their progress, scores, and upgrades when they return under the same name
- High score tracking
- Full UI — main menu, HUD, game over screen
- Custom shaders for the space environment in ShaderLab and HLSL
